## Loops

Different ways to make a loop:

### For 

```java
for (int i = 0; i < 100; ++i) {
	System.println.out(i);
}
```

### While

```java
int i = 0;
While (i <= 5) {
	++i;
}
```

### for-each

```java 
for (int n : numbers) { }
```

### Ternary
int max = (a > b) ? a : b;

### do-while
do { } while (condition);
