# `POM.xml` trong Maven

Là file cấu hình trung tâm của Maven, nó là một công cụ build và quản lý dependency phổ biến trong Java, nó chứa toàn bộ thông tin về project, plugins và cách build.

- plugins: là một mã nguồn được đóng gói riêng biệt, cho phép chương trình máy tính tương tác với nó để thêm tính năng mới mà không cần phải tay đổi hay biên dịch lại mã nguồn của chương trình gốc đó.
